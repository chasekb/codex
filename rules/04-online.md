# Online Access Policy

## Purpose
Use online sources safely, legally, and with minimal overhead.

## Requirements
1. Respect site policies (robots/TOS/rate limits) when applicable.
2. Prefer official/primary sources for technical and high-stakes claims.
3. Use minimal request volume and avoid aggressive retries.
4. Do not attempt CAPTCHA bypass or anti-bot circumvention.
5. If blocked by anti-bot/CAPTCHA, stop automation and request user guidance.
6. Provide source attribution for externally retrieved facts.

## Notes
- This policy governs behavior. Retrieval workflow stays in active tool/routing rules.
